Плагин дл wordpress для создания выкройки купальника для художественной гимнастики по введенным мерка, мерок более 30 шт отдельно включаются и выключаются присутствие руковов и юбки. после копирования в папку wp-content/plugins он появляется в меню плагинов, после активации он сосздает свои таблицы в базе wordpress и сливает туда сои данные, после активации на страницы нужно добавить шорткоды [gymnastics_pattern_form] и [gymnastics_my_patterns], первый выводит все окна ввода для мерок, второй типа личный кабинет в котором можно просматривать и корректировать выкройки которые сохранили ранее. по итогу создается файл pdf который нужно скачать и распечатать. минимальный размер выкройки A3 разбитый на 2 листа А4 для удобства, чем больше купальник тем больше листо нужно распечатать, максимальное количество листов 50, но это можно изменить в админке в соответствующей вкладке в ЛК. После установки необходимо в файлах my-pattern, admin-pattern изменить адреса страниц на которых расположены шорткоды.
